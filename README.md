專題名稱：基於 Python Socket 之多人連線遊戲 (狼人殺)
1. 環境需求
• Python 3.6 或以上版本。
• 支援 Windows / macOS / Linux 跨平台執行。
• 僅使用 Python 內建標準庫，無需安裝額外套件。
2. 檔案配置
• server.py : 伺服器端核心，負責連線、房間管理與遊戲邏輯判定。
• client.py : 客戶端程式，提供玩家互動介面與訊息收發。
3. 安裝與執行步驟
第一步：啟動伺服器 (Server)
• 開啟終端機 (Terminal) 或 CMD。
• 切換至目錄後輸入指令：python server.py
• 當顯示「📡 伺服器啟動於 0.0.0.0 6000...」即代表運作中。
第二步：啟動客戶端 (Client)
• 開啟新的終端機視窗。
• 輸入指令：python client.py
• 依照提示輸入「暱稱」即可進入大廳。
第三步：進行遊戲
• 建議至少開啟 6 個客戶端視窗模擬多人遊玩。
• 房主使用 /create 建立房間，其餘玩家使用 /join 加入。
• 全員準備就緒後，房主輸入 /start 開始遊戲。
4. 技術特點
• 多執行緒架構：實現一對多的併發連線處理。
• 數據同步機制：在關鍵變數操作處使用 threading.Lock() 確保資料正確。
• 通訊協定設計：採用 JSON 格式封裝，並透過 Buffer 與 \n 定界符解決 TCP 粘包問題。
