基於 Python Socket 之多人連線遊戲：狼人殺 (Werewolf)

1. 專案簡介：
   本系統採用主從式架構 (Client-Server)，利用 Python 內建的 Socket 與 
   Threading 技術，實作一個可多人同時在線互動的文字版狼人殺遊戲。

2. 檔案說明：
   - server.py : 伺服器端主程式，負責管理連線、房間、遊戲邏輯判定。
   - client.py : 客戶端程式，玩家執行此檔案與伺服器連線互動。

3. 環境需求：
   - Python 3.6 或以上版本。
   - 跨平台支援 (Windows, macOS, Linux)。
   - 無須安裝第三方套件 (僅使用內建 standard libraries)。

4. 安裝與執行步驟：

   第一步：啟動伺服器 (Server)
   - 打開終端機或命令提示字元 (CMD)。
   - 輸入指令：python server.py
   - 當看到「📡 伺服器啟動於 0.0.0.0 6000...」表示伺服器準備就緒。

   第二步：啟動客戶端 (Client)
   - 開啟新的終端機視窗。
   - 輸入指令：python client.py
   - 依照提示輸入您的「暱稱」。

   第三步：進行遊戲 (多人測試)
   - 重複第二步操作，啟動多個客戶端連線。
   - 玩家 A 使用指令 「/create」 建立房間。
   - 其他玩家使用指令 「/join」 加入該房間。
   - 房主輸入 「/start」 即可開始遊戲分配身分。

5. 遊戲指令參考：
   - 通用指令：
     /create - 建立房間
     /join   - 加入房間
     /leave  - 離開房間
     /who    - 查看房間內成員
     /help   - 顯示指令說明清單
   
   - 遊戲邏輯指令 (直接輸入文字)：
     投票 <名字> - 白天投票階段使用
     殺 <名字>   - 狼人夜晚使用
     查驗 <名字> - 預言家夜晚使用
     守護 <名字> - 守衛夜晚使用

6. 技術重點：
   - 多執行緒處理：每位玩家與每個遊戲邏輯皆運行於獨立 Thread。
   - 同步機制：使用 threading.Lock() 確保投票數據的一致性。
   - 資料封裝：通訊採用 JSON 格式，並透過 \n 換行符號進行封包定界。
