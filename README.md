專題名稱：基於 Python Socket 之多人連線遊戲 (狼人殺)

   - 陳永恩：上台報告與演示
   - 饒嘉豪：上台報告與演示
   指導教授：高典良 老師

2. 檔案說明：
   - server.py : 伺服器端主程式，負責管理連線、房間管理、遊戲邏輯判定。
   - client.py : 客戶端程式，玩家執行此檔案後輸入暱稱與伺服器連線。

3. 環境需求：
   - Python 3.6 或以上版本。
   - 支援 Windows / macOS / Linux。
   - 僅使用 Python 標準庫 (socket, threading, json)，無需額外安裝套件。

4. 安裝與執行步驟：

   第一步：啟動伺服器 (Server)
   - 開啟終端機 (Terminal) 或 CMD。
   - 切換至程式碼所在目錄，輸入：python server.py
   - 看到「📡 伺服器啟動於 0.0.0.0 6000...」表示伺服器運作中。

   第二步：啟動客戶端 (Client)
   - 開啟新的終端機視窗。
   - 輸入：python client.py
   - 依提示輸入「暱稱」後成功進入大廳。

   第三步：進行遊戲測試
   - 重複啟動多個客戶端以模擬多位玩家（建議至少 6 人以達到最佳遊戲體驗）。
   - 由一名玩家輸入 /create 建立房間，其餘玩家輸入 /join 進入。
   - 房主輸入 /start 即可啟動遊戲迴圈。

5. 技術亮點：
   - 多執行緒 (Multi-threading)：實現 N+1 個執行緒併發處理，確保流暢互動。
   - 互斥鎖 (Locking Mechanism)：在投票與關鍵數據寫入時使用 Lock 避免 Race Condition。
   - 通訊協定：自定義 JSON 資料格式，結合 Buffer 緩衝處理與 \n 定界解決粘包問題。
